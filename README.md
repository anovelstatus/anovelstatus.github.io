
# What is this?

This repo contains everything needed for https://anovelstatus.github.io/ to run and be powered by the community-maintained character sheet.

While there is no data in this repo, looking at the code *may spoil* information about the world of A Novel Concept.

## About A Novel Concept

You can find the story on [Royal Road](https://www.royalroad.com/fiction/66455), and support it on [Patreon](https://www.patreon.com/ANovelConcept).

Join the community on [Discord](https://discord.gg/HQDDnwpFmS) to discuss the latest chapters.

## Code Layout

- [website/](website/) contains the source that is built for the GitHub Pages site linked to this repo.
- [shared/](shared/) contains TS types used by both the website and the API.
- `api/` will eventually contain the source for Apps Script project serving as the API
